Environment Setup

To use our build system and run our documentation locally, you’ll need a copy of Tabler’s source files. Follow the steps below:

  1. install package dependecies via yarn or npm

  2. In the comand line run gulp

  3. open in http://localhost:3000/ in your browser

  4. Any change in /src directory will build the aplication and refresh the page
