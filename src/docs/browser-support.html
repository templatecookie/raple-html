As of this release, Relik has launched many new features that may bot be compatible with all browsers, especially older ones. For instance, Relik no longer uses vendor prefixes in styles that are known to have some compatibility issues except for a few parts. Relik also supports new features such as CSS grid, object fit, and sticky positioning which is not yet supported on all browsers. In short, Relik supports all the latest supported browsers with only partial support for Internet Explorer.
