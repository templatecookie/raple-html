Layouts

Verticle Sidebar

By Default if You want to set verticle sidebar , you have to add attribute ( data-layout ) inside html. attributes value would be verticle, means data-layout="verticle". That's it

Horizontal Sidebar

By Default if You want to set verticle sidebar , you have to add attribute ( data-layout ) inside html. attributes value would be verticle, means data-layout="horizontal". That's it

Themes

Defaul Theme

By default you will get this theme. So you have not to do anything. if you didn't like default theme then you can checkout other's. because Wadmin will give you ton's of good options, that's will help you to decide.

Dark Theme

By Default if You want to use Dark Theme, you have to do something with your html file . Go on html file and add bellow attribute to your html tag. data-sidebar="black"

Navy Theme

By Default if You want to use Dark Theme, you have to do something with your html file . Go on html file and add bellow attribute to your html tag. data-sidebar="blue"
