Map

Relick uses jVectorMap - a free and open-source modern map library that helps developers to create beautiful and interactive visualizations for web pages.

Usage

to able to use jvector you have to link css and js files ( jquery-jvectormap-2.0.5.css, jquery-jvectormap-2.0.5.min.js, jquery-jvectormap-world-mill-en.js )

Calendar

Relick uses javascript FullCalendar - a free and open-source modern calendar library that helps developers to create beautiful and interactive visualizations for web pages.

Usage

before you implement Full calendar you have to link those files that are mentioned below. all of those files you will get from assets/css & assets/js

  • Css

  • @fullcalendar/core/main.min.css

  • @fullcalendar/daygrid/main.min.css

  • @fullcalendar/bootstrap/main.min.css

  • @fullcalendar/timegrid/main.min.css

  • Js

  • @fullcalendar/core/main.min.js

  • @fullcalendar/bootstrap/main.min.js

  • @fullcalendar/daygrid/main.min.js

  • @fullcalendar/timegrid/main.min.js

  • @fullcalendar/interaction/main.min.js
